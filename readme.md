 LebaneseAirline - Travel Booking System

Frontend prototype of a flight booking system built with:

- PHP
- HTML
- CSS
- JavaScript
-JSON (temporary data storage for authentication)

## Features

- Flight search
- Booking system UI
- User dashboard
- My bookings page
- Profile management
- Authentication pages
- Admin dashboard interface
